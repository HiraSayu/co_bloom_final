##READ ME
このリポジトリはwaffle colleage修了制作で作ったwebアプリ「co-bloom」、の
HTML・CSS・javascript部分のみをアップロードしています。
firebaseとの紐付けやgsutilの設定などが行われていないため、アプリを使用することはできません。
公開されているページはこちら：https://co-bloom.firebaseapp.com/
